what is dom and what is the purpose of dom?

DOM(Documents Object Model)is a programing interface that can access documents,can change document structure,style and content also. DOM is used to "speeding up web page loads", access documents,change document structure,style and content.

How will you select HTMl elements using DOM?Name the DOM methods

I can select HTML elements using DOM methods. DOM methods are :

  • document.getElementById()
  • document.getElementsByTagName()
  • document.getElementsByClassName()
  • document.querySelector()
  • document.querySelectorAll()

What is event bubble?

Event Bubbling is a concept in the DOM (Document Object Model),when an elements receives and event target consecutively triggers the parent element of the target element in the same hierarchy until it triggers the outermost element.

What is a callback function and why will you use it?

A callback function is a function which value can be used as a argument of another function. I use a callback function to avoid to write same function.
